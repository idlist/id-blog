大语言模型（LLM）自从广泛面世以来，带给我们的不止是惊喜，还有更多的惊吓。这或许是人类史上最强力的工具之一，强力到我们自己也并没有能够摸索出合理的用法。

在这篇文章里，我不想谈 LLM 造成的深度伪造（Deep Fake）和信息泥沼（AI Slop）问题，我也不想谈 AI 取代人或者成为压榨人的工具的问题。我们来说一些更难以发现，但更加本源的问题。

## 成瘾性

早十年前说到这个话题，大家都还是嗤之以鼻；现在提起这个话题，大家不再陌生了。从电子游戏，到社交媒体，再到短视频，最后到目前的 LLM，人们一步步地创造出能够更有效率地用多巴胺轰炸自己大脑的造物。在电子游戏这一档，它的成瘾性还算相对可控，但是从社交媒体往后，它们的成瘾性及其长期不良后果已经由众多的学术研究所佐证。除了目前被低估的 LLM。

就目前人们对于 LLM 的商业运用模式而言，我可以自信地把 LLM 放在这个链条的延长线上。想想目前的模式：用户输入一段话，大模型输出一长串文章、一整张图片甚至一小段视频给你。你可以把这个叫做“生产力的提升”，但这种和 AI 的**交互过程**本身就是具有高度**成瘾性**的。实际上，这和你打开推特刷新推文，或者打开抖音刷新视频并没有太大的区别，唯一说得上区别的是内容的入口变成了你的一句话——而这其实更**糟糕**。以前你至少还需要去关注人，然后推荐算法也至少得从你的互动去猜你喜欢什么，而现在，就连这些（即使微小但也还是必要的）“努力”也都被跳过了。

然后，LLM 的特性导致它还和另一个恶魔交上了朋友：**变比率强化程序**，也就是**赌博**的成瘾机制。试想，同样的提示词，这一次的文章看起来不怎么样，但或许点一下重新输出，就能获得更好的文章；同样的提示词，这一次的视频看起来像是儿童邪典，但下一次的就是科幻大片——这种类似于抽卡的不确定性，可以说是 LLM 的魅力，也可以说是 LLM **最糟糕**的地方。

## 能动性的下降

这其实是前面这一点带来的后果之一，而我自己深有体会。大约一两年前，LLM 还没有像当今这么成熟的时候，我还能够比较轻松地起草并写出像样的短篇文章；但是现在，在 LLM 的熏陶下，我**自身**的成文能力肉眼可见地**下降**了。我写东西还是姑且有一点个人风格在里面，因此在写东西上，我从来没找 LLM 帮过忙。但**即使如此**，我的成文能力也**仍然**下滑了。

解释这个原因其实不难。在 AI 聊天的场景下，用户输入几个字、几句话，LLM 就能输出一大段字喂给大脑；而靠自身能力成文的时候，大脑绞尽脑汁才能蹦出几句话。换句话说，在这两种场景下，大脑付出的努力是**完全无法比较**的。久而久之，大脑在面对成文任务的时候就会产生**惰性**：为什么我要自己动？尤其是明明在旁边就有一个更省力的路径的情况下。

惰性还不是唯一的问题。还有另一个问题如影随形：**文章风格的均质化**。仍旧拿写作举例，在不过度考虑商业化的前提下，这个行为理应首先是**个性**的展现，然后再是其他。它是一个人的风格、见识、态度，或者说大脑神经元的连接结构极其内部储存的内容的独特表现。而在当前这种和 LLM 的交互下，大脑输入了大量的由 LLM 生成的东西，在这个具有成瘾性的、强烈的反馈循环中，大脑也逐渐变成 LLM 的风格了，久而久之也就只能写出 LLM 风格的内容了，而到这一步也不用我来说了：这很糟糕。

## 后记

乐观地看，这个情况姑且是一时的，因为和 LLM 交互的形式还有充足的改良余地。就我个人的观点而言，LLM 应当回归到“好用的工具”上，在它的工具属性上做进一步的开发；而这种不对称的暴力内容输出永远不应该是 LLM 的长久商业形式。

但是总所周知，事情并不会就这么简单地你好我好大家好了。如果 LLM 依旧按当今的路径继续蚕食人们的注意力、虐待大脑的信息处理极限，那么恐怕“信息污染”也将需要像生物、化学行业那样，迈入被监管的范畴。当然，这大概又是另一种妄想。
