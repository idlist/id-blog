# i'D Blog

从前有一个人，他有个自己写静态博客生成器的想法，但他写不出来。

## 准备做的

- [x] 首页
- [x] 文章分页
- [x] 文章
- [x] 标签分类
- [ ] 标签分类分页（现在在这里）
- [x] 时间分类
- [ ] 时间分类分页

## 效率

热重载下初次启动 1 秒，更新文件大约 0.5 秒。

## 实现了什么

- 用 TypeScript 写模板和网页脚本。
- 用 Sass 写样式。
- 模板嵌套。
- 热重载，虽然只是主动刷新网页。
- 色彩丰富的控制台输出。

## 没有实现什么

- 解耦。基本所有的构建过程都是高度耦合的。
- 框架化。因为没能解耦。
